## 🌌 SECTION 1 — CINEMATIC PARALLAX HOMEPAGE (`index.html`)

Build a multi-layered, hardware-accelerated CSS parallax engine:

**Layer 1 — Deep Background Canvas**
Clean `#0D0A14` foundation with fluid, dreamy ultra-soft purple HSL gradients and ambient cloud shapes.
```css
transform: translateZ(-2px) scale(3);
```
Interactive: shapes drift slowly on `mousemove` at 0.02× cursor offset ratio.

**Layer 2 — 3D Midground Space**
High-fidelity claymorphic 3D-style CSS models: premium gift boxes, floating celebratory packages, glossy coin shapes — floating at independent scroll speeds.
```css
transform: translateZ(-1px) scale(2);
```

**Layer 3 — Foreground Interactive UI**
High-contrast glassmorphic panels:
```css
backdrop-filter: blur(12px);
background: rgba(255, 255, 255, 0.04);
border: 1px solid rgba(255, 255, 255, 0.08);
border-radius: 24px;
```
Tracks at normal scroll speed. Hosts all primary text and CTAs.

**Hero Copy:**
- Main Serif Headline (Cormorant Garamond, 72px–96px): `"Gift the Feeling. Celebrate in Tamil Style."`
- Sub-headline (DM Sans, 18px–22px): `"Curated return gifts, engraved keepsakes, and celebration hampers — built for every occasion, delivered with cultural pride."`
- Search field placeholder: `"Search gifts, occasions, or collections..."`
- Primary CTA button: `Order & Inquire`
- Secondary ghost CTA: `Explore Collections`

**Hero layout:** Full viewport height (`100dvh`). Text left-aligned on desktop, centered on mobile. Dual CTA row. Subtle scroll-progress indicator (thin Kancheepuram Gold line at bottom).

### Flash Sale Renderer (critical — exact logic):
```js
const { flashSaleActive, endsAt } = config
if (!flashSaleActive) { container.style.display = 'none'; return }
const targetTime = new Date(endsAt).getTime()
if (Date.now() >= targetTime) { container.style.display = 'none'; return }
// Render countdown, store interval ID in container.dataset.intervalId for cleanup
const interval = setInterval(() => {
  const remaining = targetTime - Date.now()
  if (remaining <= 0) { clearInterval(interval); container.style.display = 'none'; return }
  const d = Math.floor(remaining / 86400000)
  const h = Math.floor((remaining % 86400000) / 3600000)
  const m = Math.floor((remaining % 3600000) / 60000)
  const s = Math.floor((remaining % 60000) / 1000)
  document.querySelector('#cd-days').textContent  = String(d).padStart(2,'0')
  document.querySelector('#cd-hours').textContent = String(h).padStart(2,'0')
  document.querySelector('#cd-mins').textContent  = String(m).padStart(2,'0')
  document.querySelector('#cd-secs').textContent  = String(s).padStart(2,'0')
}, 1000)
container.dataset.intervalId = interval
```

## 🔄 SECTION 6 — THEME LOADER (`theme-loader.js`)

Load in `<head>` as non-deferred. Runs before DOMContentLoaded. Every HTML file needs `<style id="mz-theme-vars"></style>` in `<head>`.

**Execution flow:**
1. Check `sessionStorage["mz-theme-v{version}"]` for cached settings. If version matches ETag, skip fetch.
2. `GET /api/site-settings/theme` — on network failure, load `theme.defaults.css` values.
3. Read `SETTINGS_TO_CSS_MAP` (MongoDB key → CSS variable, single source of truth at top of file).
4. Iterate map, build CSS custom property declaration string. Use defaults for any undefined values.
5. Write: `document.getElementById('mz-theme-vars').textContent = \`:root { ${declarations} }\``
6. Append Google Fonts `<link>` tags with `display=swap`. Never add duplicates.
7. Listen for `CustomEvent("mz:theme-updated")` → re-run steps 2–7 without page reload.
8. Cache in `sessionStorage` with version key.

SECTION 3 — THEME LOADER (theme-loader.js)
════════════════════════════════════════════════════════════
This script runs before DOMContentLoaded — load it in <head> as a
regular (non-deferred) script so vars are present at first paint.

Execution flow:

1. Attempt to read cached settings from sessionStorage key "mz-theme-v{version}".
   If present and version matches the ETag from the last GET, skip fetch.

2. Fetch GET /api/site-settings/theme
   On network failure: load theme.defaults.css values as the fallback.
   Never surface a broken layout to the user.

3. Build the SETTINGS_TO_CSS_MAP constant:
   This is the single source of truth for every MongoDB key → CSS variable mapping.
   Format: { "theme.hdr.bg": "--hdr-bg", "theme.hdr.border": "--hdr-border", ... }
   Include ALL variables from all 16 groups + all layout tokens.
   This map lives at the top of theme-loader.js. Adding a new var = adding one line here.

4. Iterate the map. For each key: read value from settings payload.
   If value is undefined or empty: read from theme.defaults.css fallback map.
   Build a single string of CSS custom property declarations.

5. Write into:
   document.getElementById('mz-theme-vars').textContent = `:root { ${declarations} }`

   The <style id="mz-theme-vars"> tag must exist in every HTML file's <head>
   (empty on server render, populated by this script).

6. For each typography role: append Google Fonts <link> tags to <head>.
   Always add display=swap. Always provide the fallback stack.
   Never add duplicate link tags (check before appending).

7. Write typography and layout tokens into #mz-theme-vars as additional
   CSS custom properties on :root.

8. Listen for CustomEvent "mz:theme-updated" on window.
   On fire: re-run steps 2–7 without page reload (for live preview).

9. Store settings in sessionStorage with version key for cache.

    if (remaining <= 0) {
      clearInterval(interval)
      container.style.display = 'none'
      return
    }
    const d = Math.floor(remaining / 86400000)
    const h = Math.floor((remaining % 86400000) / 3600000)
    const m = Math.floor((remaining % 3600000) / 60000)
    const s = Math.floor((remaining % 60000) / 1000)
    document.querySelector('#cd-days').textContent  = String(d).padStart(2,'0')
    document.querySelector('#cd-hours').textContent = String(h).padStart(2,'0')
    document.querySelector('#cd-mins').textContent  = String(m).padStart(2,'0')
    document.querySelector('#cd-secs').textContent  = String(s).padStart(2,'0')
  }, 1000)

  // Store interval id for cleanup on SPA navigation
  container.dataset.intervalId = interval
